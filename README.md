# 📈 Price Tracker
By Hussain Farhan

Scrapes product prices across multiple pages and exports to CSV with daily timestamps.

## What it does
- Multi-page scraping with automatic pagination
- Extracts title, price, rating, availability
- Saves to price_data.csv

## How to run
pip install requests beautifulsoup4
python price_tracker.py

## Tech used
Python, Requests, BeautifulSoup4

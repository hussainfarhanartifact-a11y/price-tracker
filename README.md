# Price Tracker
**By Hussain Farhan**

Scrapes product titles, prices, ratings and availability across multiple pages and exports to a clean CSV file with daily timestamps.

## What it does
- Loops through multiple pages automatically
- Extracts title, price, star rating, availability
- Timestamps every scrape with today's date
- Saves to price_data.csv

## How to run

Install dependencies:
pip install requests beautifulsoup4

Run it:
python price_tracker.py

## Output
| title | price | rating | availability | date_scraped |
|-------|-------|--------|--------------|--------------|
| A Light in the Attic | £51.77 | Three | In stock | 2024-01-01 |
